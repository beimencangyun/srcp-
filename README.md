Q:为什么进去存档提示缺少模组
A:因为需要加载雕凿工艺模组才能进入游戏

Q:为什么必须要加载雕凿工艺模组
A:因为需要更好的完善建筑细节(比如公路的路牌)

Q:为什么进入游戏看不到城市
A:城市原点坐标0 64 0(找不到城市可以加装地图模组)

Q:为什么使用minecraft 1.12.2过时的版本
A:历史原因 1因为开地图时就是这个版本无法更改 2因为这个版本很流畅 3这个版本使用很舒服

Q:为什么不是最新版本而是删减后的版本
A:因为避免不必要的纠纷 如果你想体验新版本请发邮箱
